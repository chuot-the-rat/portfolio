### Adding Hover Images

The component automatically extracts hover images from your project data:
- First 2 images from `solution.images` (after the main image)
- Plus 1 image from `overview.images`

Or explicitly define them:
```json
{
  "hoverImages": [
    { "src": "/path/to/hover1.png", "alt": "Detail view" },
    { "src": "/path/to/hover2.png", "alt": "Close-up" }
  ]
}
```

## ğŸ¯ Component Deep Dive

### ProjectCard.jsx

Premium hover interactions with Framer Motion:

```jsx
<motion.article
  whileHover={{ 
    y: -8,
    scale: 1.02,
    transition: { duration: 0.3, ease: [0.4, 0, 0.2, 1] }
  }}
>
  {/* Main image with zoom */}
  <motion.div animate={{ scale: isHovered ? 1.05 : 1 }} />
  
  {/* Hover-to-reveal images with stagger */}
  {hoverImages.map((img, i) => (
    <motion.div
      animate={{ opacity: isHovered ? 1 : 0 }}
      transition={{ delay: i * 0.1 }}
    />
  ))}
  
  {/* Text fade-in */}
  <motion.h3 animate={{ opacity: isHovered ? 1 : 0.7 }} />
</motion.article>
```
